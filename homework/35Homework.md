3. Computed 속성과 watch 속성에 대하여 간단하게 설명하고, 이 둘의 차이점에 대해
작성하시오.

```
computed는 감시하고 있는 값이 바뀌는 것을 인식하면 값을 자동으로 계산해서 바꿔준다.
즉 값이 변하지 않으면 내부에 정의된 함수를 호출하여도 다시 계산하지 않고 캐싱한 결과를 즉시 반환해준다.

watch는 computed와 마찬가지로 특정 값을 감시하고 있다. 그리고 값이 바뀌면 함수를 실행한다. 여기서 computed와 watch의 차이점은 computed는 값이 바뀔 때 새로운 계산을 수행해서 결과 값으로 종속 대상을 바꿔주는 반면 watch는 값이 바뀔 때 함수를 실행하는 것이다. 따라서 값 변경 외에 다른 목적으로 함수를 실행한다면 watch를 쓰는 것이 유리하고 값 변경 때마다의 새로운 계산 값을 얻기 위해서는 computed를 사용하는 것이 맞다. 대부분의 경우 computed를 사용할 것이다.
```
