#### 1. JavaScript에서 자주 사용하는 Event Listener에는 아래와 같은 것들이 있다. 각각 어떤
Trigger를 의미하는지 조사하여 간단하게 작성하시오.
* **click** :  마우스 클릭
* **mouseover** :  마우스를 올리면
* **mouseout** :  마우스를 올렸다 떼면
* **mousemove** :  마우스를 움직이면
* **keypress** :  키보드를 누르는 동안
* **keydown** :  키보드를 누르는 순간
* **keyup** :  키보드를 떼는 순간
* **load** :  창을 키면
* **scroll** :  스크롤을 움직이면
* **change** :  element값이 변경되면
